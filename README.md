# HerSignal

**AI-Powered Safety Companion for Women**

## üöÄ Getting Started

### Web Platform Development

#### Prerequisites
- Node.js 18+ 
- npm or yarn

#### Installation
1. Clone or download the project
2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

#### Build for Production
```bash
npm run build
npm run lint  # Check for any issues
```

### iOS App Development

#### Prerequisites
- macOS Ventura+ 
- Xcode 15+
- Apple Developer Account ($99/year)
- Basic Swift knowledge

#### Getting Started
1. Review the complete `Swift.md` guide
2. Create new Xcode project following the structure outlined
3. Implement core features step-by-step
4. Test with target users in realistic scenarios
